# Eleventy Blog

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Build for production:
   ```
   npm run build
   ```

## Project Structure

- `src/` - Source files for your blog
  - `_includes/` - Reusable templates and layouts
  - `posts/` - Blog post markdown files
  - `css/` - Stylesheets
  - `js/` - JavaScript files
  - `images/` - Images and media
  - `index.md` - Home page
- `_site/` - Generated static site (created during build)
- `.eleventy.js` - Eleventy configuration

## Getting Started

1. Add blog posts to `src/posts/` as markdown files
2. Create layouts in `src/_includes/`
3. Customize the configuration in `.eleventy.js`
4. Add your CSS to `src/css/`
